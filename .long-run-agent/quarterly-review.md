## 🔍 现行规则评估

### NON_NEGOTIABLE

- [ ] **交付物必须存在**：仍必要？
- [ ] **覆盖率 ≥ 70%**：阈值是否合适？调整？
- [ ] **L2 零依赖**：是否仍执行？
- [ ] **私钥加密**：所有私钥都加密？
- [ ] **TLS 1.3**：生产 100% 启用？
- [ ] **审计日志**：所有写操作记录？

### MANDATORY

- [ ] **AAP 鉴权先行**：所有写路径？
- [ ] **结构化日志**：JSON 格式 100%？
- [ ] **Prometheus 指标**：覆盖率 100%？
- [ ] **OTel trace**：跨服务传播正常？
- [ ] **Conventional Commits**：执行率？
- [ ] **CHANGELOG 更新**：每个 PR？

### CONFIGURABLE

- [ ] **限流阈值**：当前值是否合理？
- [ ] **缓存 TTL**：命中率？
- [ ] **存储后端**：生产使用？
